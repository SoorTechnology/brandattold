Stock Inventory Adjustments Import Using CSV

Stock Inventory Adjustments Import App for Odoo.

This app allow you to import stock inventory adjustments from csv file.

Main Features:

  • This app will import stock inventory adjustments using csv.
  • You have to prepare CSV file with Product Code / Product Name / Product Barcode, Real Quantity, Unit of Measure, Lot/Serial Number, Pack, Owner and import it.
  • Sample CSV File is available inside sample folder in app.
  • For more details see video and below screenshots.

Following fields can be Import From CSV Column.

  • Product Name/ Product Code/ Product Barcode : Enter Product name or Product Code or Product Barcode.
  • Real Quantity : Real Quantity for Product.
  • Unit of Measure : Unit of Measure of Product.
  • Lot/Serial Number : Lot/Serial Number for Product.
  • Pack : Package Number (Optional).
  • Owner : Owner Name (Optional).
